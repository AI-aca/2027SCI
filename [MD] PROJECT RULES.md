## 🤖 AI 채점 모드 스위칭 규칙 (일반 모드 vs 깐깐 모드)
사용자가 대화 중에 채점 모드 변경을 요청하면, AI는 즉시 Code.gs의 systemPrompt 내 '[🚨 1순위 규칙: 보수적(엄격한) 정성평가 및 횟수 카운팅 기준 (거품 제거)]' 블록을 켜거나 끕니다.
- **일반 모드 (Normal Mode)**: 해당 1순위 엄격 규칙을 삭제합니다. (학생의 긍정적 측면을 폭넓게 인정하여 점수대가 비교적 높게 나옴)
- **깐깐 모드 (Strict Mode)**: 해당 1순위 엄격 규칙을 주입합니다. (동아리/탐구 카운트를 극도로 보수적으로 산정하고, 세특/행특 칭찬 멘트를 강등시켜 점수 거품을 제거함)
- 사용자가 "일반 모드로 바꿔줘" 또는 "깐깐 모드로 바꿔줘"라고 말하면, 별도의 설명 없이 즉각 Code.gs 파일을 패치하여 프롬프트를 교체하고 배포 가이드를 안내합니다.
