# [2027 과학고 관리 사이트] 역할별 시스템 사용 매뉴얼

> 본 문서는 학생, 강사, 관리자 각각의 권한에 따라 화면에 다르게 노출되는 맞춤형 사용 매뉴얼(가이드)의 원본 초안입니다.
> **사용자의 승인 후, 이 문서의 내용이 시스템(`script.js`)에 실제 렌더링 코드로 반영됩니다.**

---

## 👨🎓 학생 전용 매뉴얼 (학생 로그인 시 단독 노출)

**학생 여러분, 환영합니다!**
본 시스템은 여러분의 2027년 과학고 합격을 위한 자기소개서 작성 및 이력 관리를 돕기 위해 제작되었습니다. 아래 가이드를 읽고 시스템을 활용해 주세요.

### 1. 자기소개서 작성 및 관리
* **맞춤형 문항 자동 배정:** 여러분이 지원하는 학교(예: 인천과학고, 진산과학고 등)에 맞춰, 자소서 탭 진입 시 자동으로 해당 학교의 문항과 글자 수 제한이 배정됩니다.
* **글자 수 실시간 체크:** 자소서 편집창 하단의 글자 수는 **공백을 포함**하여 실시간으로 계산됩니다. 지정된 글자 수를 넘지 않도록 주의하세요.
* **수시 임시 저장:** 작성 중 서버 오류나 브라우저 종료로 인한 데이터 날아감을 방지하기 위해, 수시로 `[임시 저장하기]` 버튼을 눌러주세요.
* **최종 제출 주의:** 작성이 완전히 끝나고 `[최종 제출하기]`를 누르면 자소서가 **잠금 처리되어 더 이상 수정할 수 없게 됩니다.** 신중히 클릭해 주시고, 부득이하게 다시 수정해야 한다면 관리자 선생님께 락(Lock) 해제를 요청해야 합니다.

### 2. 선생님의 수기 첨삭 및 AI 피드백 확인
* **피드백 조회:** 자소서 편집기 하단의 `[수기 피드백]` 탭을 누르면, 담당 선생님이 작성해주신 꼼꼼한 첨삭 지도를 실시간으로 확인할 수 있습니다.
* **AI 평가 확인:** 선생님께서 AI 분석을 실행해주셨을 경우, 자소서에 대한 AI 평가 코멘트도 함께 확인하여 글의 방향성을 잡을 수 있습니다.

### 3. 면접 예상질문 직접 답변하기 (★중요)
1. 상단의 **[면접 연습]** 탭으로 이동합니다.
2. 본인의 이름과 지원 학교가 적힌 행 우측 끝의 `[본인 면접 답변하기]` 버튼을 클릭합니다.
3. 시스템이 여러분의 생기부와 자소서를 바탕으로 뽑아준 맞춤형 면접 예상질문 리스트가 뜹니다.
4. 질문 우측의 텍스트 박스에 해당 질문에 대한 나만의 모범 답변을 직접 타이핑하고 `[답변 저장]`을 누르며 면접 대비를 할 수 있습니다.

### 4. 챗봇 'SION' 활용 가이드
* 화면 우측 하단의 비서 **SION** 채팅창을 통해 과학고 입시, 면접 팁, 자소서 작성 방향에 대해 언제든지 자유롭게 질문해 보세요.

## 👨🏫 강사 전용 매뉴얼 (강사 로그인 시 단독 노출)

**강사님, 수고 많으십니다.**
본 시스템은 담당 학생들의 자기소개서 첨삭, 면접 지도 및 진학 관리를 효율적으로 수행할 수 있도록 돕는 강사 전용 기능들을 제공합니다.

### 1. 학생 상태 관리 및 생기부 열람
* **합불 상태 갱신:** 대시보드(초기화면)의 '1차 합격', '2차 합격' 열을 클릭해 학생의 전형 결과를 실시간 드롭다운으로 변경할 수 있습니다.
* **학생 정보 수정:** 학생 목록 맨 우측 관리 열의 `[수정]` 버튼을 눌러, 연락처를 수정하거나 담당 강사(수학/과학)를 재배정할 수 있습니다.
* **생기부 열람 및 3개 영역 요약 모달:**
  * 대시보드에서 학생의 **생기부 점수 수치**를 클릭하면, AI가 분석한 생기부의 **[3개 핵심 영역(학업역량, 진로적합성, 인성) 요약본 모달]**이 열립니다.
  * 점수 옆의 **PDF 아이콘**을 클릭하면, 학생의 원본 생활기록부 파일을 즉시 브라우저에서 열람하거나 새 PDF로 덮어쓰기 업로드할 수 있습니다.

### 2. 자기소개서 수기 피드백 지도
1. 상단 메뉴의 **[자기소개서]** 탭으로 이동하여 지도할 학생을 클릭합니다.
2. 우측 화면 하단의 **'수기 피드백'** 탭을 엽니다.
3. 강사님의 꼼꼼한 지도 의견, 보완해야 할 방향 등을 작성하고 `[피드백 저장]`을 누르면 학생 화면에 즉시 공유됩니다.
*(참고: 자소서를 기반으로 한 자동 AI 피드백 생성 버튼은 관리자 계정에서만 실행 가능합니다.)*

### 3. 면접 답변 확인 및 지도
* 상단 메뉴의 **[면접 연습]** 탭에서 특정 학생 행 우측의 `[답변 확인]` 버튼을 클릭하십시오.
* 학생이 직접 타이핑한 면접 질문별 답변 내용을 열람할 수 있으며, 이를 바탕으로 구두 모의 면접을 지도하시면 됩니다.

## 👑 관리자 전용 매뉴얼 (관리자 로그인 시 단독 노출)

**최고 관리자님, 환영합니다.**
본 계정은 강사가 수행하는 모든 기능(수기 피드백, 합불 갱신 등)을 기본적으로 사용할 수 있으며, 시스템의 근간을 제어하는 마스터 권한이 추가로 부여됩니다.

### 1. 생기부 정밀 분석 (23개 전체 항목) 열람
* 대시보드에서 **생기부 점수 수치**를 클릭하면, **[23개 상세 항목 정밀 분석 모달]**이 열립니다.
* 강사 계정에서는 3개 영역 요약본만 표기되지만, 관리자 계정에서는 23개 평가 요소 전체의 개별 세부 점수와 AI가 그렇게 산정한 구체적 판단 근거가 기재된 표를 전체 열람할 수 있습니다.

### 2. 학생 일괄 AI 자동화 처리 (자소서 / 면접)
* **자소서 AI 일괄 첨삭:** [자기소개서] 탭 상단의 `[일괄 AI 피드백 생성]` 버튼을 통해, 다수의 학생 자소서를 Gemini API를 통해 일괄 분석시킬 수 있습니다.
* **면접 예상질문 일괄 생성:** [면접 연습] 탭 상단의 `[일괄 AI 예상질문 생성]` 버튼을 눌러, 여러 학생의 자소서 기반 맞춤형 예상질문을 한 번에 추출하십시오.

### 3. 학생 자소서 최종 제출 '락 해제' 권한
* 학생이 자기소개서 작성을 마치고 '최종 제출' 버튼을 누르면 문서가 잠금(Lock) 처리됩니다.
* 수정을 다시 허가해야 할 경우, 관리자 계정으로 해당 학생의 자소서 편집창을 열고 내부의 `[최종 제출 락 해제]` 버튼을 눌러 락을 풀어줄 수 있는 **유일한 승인 권한**을 가집니다.

### 4. 시스템 마스터 환경 설정 제어
* 사이드바 하단의 **[설정]** 메뉴에 진입하여 다음 시스템 핵심 정보들을 실시간으로 관리할 수 있습니다.
  * **보안:** 관리자 및 강사 로그인 비밀번호 변경
  * **API:** 구글 Gemini API 연동 키 업데이트
  * **저장소:** 자소서, 생기부, 피드백, 입학요강 등 각 기능별 구글 드라이브 폴더 ID 연결 관리
  * **모집요강:** 각 과학고별 1~4번 문항 텍스트와 개별 글자수 제한(Byte/자) 실시간 수정 및 반영
